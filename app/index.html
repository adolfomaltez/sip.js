Escenarios

The SIP.js library and the demos must be built before they will run. 

    npm install
    npm run build-demo
Safari requires either...
  • enabling Develop -> WebRTC -> Allow Media Capture on Insecure Sites
  • or serving the demo from a secure website

1) Caso # 1: Usuario hace login a la aplicacion WEB y se registra su extension en el servidor Asterisk.

  • Connecting to SIP WebSocket Server
  • Making an outbound audio call
  • Sending DTMF
  • Hold & Mute

2) Caso # 2: Usuario hace llamada a aplicacion IVR en Asterisk.

  • Connecting to SIP WebSocket Server
  • Registering to receive incoming calls
  • Making an outbound video call
  • Answering incoming calls

3) Caso # 3: Usuario llama a otro Usuario WEB en la misma planta Asterisk.

  • Connecting to SIP WebSocket Server
  • Registering to receive incoming calls
  • Making an outbound call with data channel
  • Answering incoming call with data channel

4) Caso # 4: Usuario llama a usuario de Red PSTN. (Telefono celular o fijo).

  • Connecting to SIP WebSocket Server
  • Registering to receive incoming calls
  • Making an outbound call with data channel
  • Answering incoming call with data channel

5) Caso # 5: Usuario recibe llamada (y metadata) de red PSTN.

  • Connecting to SIP WebSocket Server
  • Registering to receive incoming calls
  • Making an outbound call with data channel
  • Answering incoming call with data channel

6) Caso # 6: Usuario entra en conferencia con 2 o mas usuarios.

  • Connecting to SIP WebSocket Server
  • Registering to receive incoming calls
  • Making an outbound call with data channel
  • Answering incoming call with data channel
